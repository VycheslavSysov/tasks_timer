# Tasks Timer

SPA-застосунок для обліку часу, витраченого на задачі.

## Функціонал

- запуск і зупинка таймера для задачі
- збереження задач у лог
- перевірка на порожню назву задачі з модальним попередженням
- збереження стану після перезавантаження сторінки
- продовження таймера після reload
- вкладка з логом задач
- вкладка з графіком розподілу часу по годинах
- сторінка деталей задачі
- генератор тестових задач
- Jest-тести для утиліт

## Технології

- React
- Vite
- Redux Toolkit
- React Router
- Recharts
- Tailwind CSS
- Jest

## Запуск проєкту

Встановити залежності: npm install

Запустити dev-сервер: npm run dev

Запустити тести: npm test

Перевірити lint: npm run lint

Зробити production build: npm run build

## Збереження даних

Задачі та стан таймера зберігаються в `localStorage`, тому після перезавантаження 
сторінки дані не втрачаються.
